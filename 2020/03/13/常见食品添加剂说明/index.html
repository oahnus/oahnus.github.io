常见食品添加剂说明

发表于 2020-03-13 更新于 2020-03-25 分类于 杂货柜

A

阿斯巴甜

非碳水化合物类人造==甜味剂==。甜度高，热量低，主要用于饮料，含片代替糖使用。高温会使其分解失去甜味，不适用于烹饪和热饮。

安赛蜜

乙酰磺胺酸钾，一种无热量==甜味剂==。甜度是蔗糖的200倍，和阿斯巴甜相当，是糖精的三分之二，三氯蔗糖的三分之一。有较好热稳定性，可用于烘焙。

C

呈味核苷酸二钠

同5’-呈味核苷酸二钠，==增鲜剂==。主要用于肉类加工，鱼类加工，饮食业等

D

D-异抗坏血酸钠

食品行业作为抗氧化剂，==防腐剂==。能延长保质期，无毒副作用。

J

焦亚硫酸钠

主要用作==漂白剂==，消毒剂，==防腐剂==。可用于食物的漂白和防腐，用量有严格限定，对人体有一定损伤。经过焦亚硫酸钠加工的食品，使用50度温水经过浸泡洗净2次，可以降低含硫量约92％。

N

柠檬酸

自然在柑橘类水果中产生的一种==天然防腐剂==，也是食物和饮料中的==酸味添加剂==。

柠檬酸钠

可用作==酸度调节剂==，缓冲剂、乳化剂、稳定剂。安全无毒。

S

三氯蔗糖

人造==甜味剂==，甜度可达蔗糖的600倍。无热量，甜度高，安全度高。但蔗糖素在食品的加工过程中可能（受热）分解，并产生对人体有害的毒性产物。

山梨酸钾

主要用于==食品防腐剂==，也可用于化妆品防腐和饲料防霉。有很强的抑制腐败菌和霉菌作用，又易溶于水和毒性极低而获得广泛应用。

羧甲基纤维素钠

==增稠剂==和==乳化剂==，对含油脂蛋白质的饮料可起到乳化稳定作用。常用于乳制品。

T

特丁基对苯二酚

抗氧化剂和==防腐剂==。用于油脂和含油脂食品中。耐高温。能有效抑制细菌及霉菌生长。长期服用含有特丁基对苯二酚的食品可能致畸或致癌（存在争议）。
